Information set up the printer on Maths PHD students using linux.


On the task bar click

System -->  Administration --> Printing

Enter root password

In Printer Configuration Box click the New button

Click Forward

For name enter: LaserJet  then click forward.

The Queue Type option change from locally - connected to Networked JetDirect
And then for printer: 138.40.164.108 port: 9100   (printer number error, corrected by E Banjo)

Then Click Forward

In printer model scroll down to PostScript Printer in the Generic section (don't select HP)

Thats your printer set up!
