The 2-dimensional isoperimetric inequality - an elementary proof

The isoperimetric inequality asserts that under all given closed curves of a fixed lenght in the plane, the enclosed region has maximal area if and only if the closed curve is actually a circle. There is a nice and short proof of this using only some basic facts about Fourier series.

The project has two steps:
I. Proof of Leibniz' sector formula; this is elementary analysis.
II. Proof of the isoperimetric inequality.

Books

H. Dym and H.P. McKean, Fourier series and integrals, Academic Press, 1972.
E.M. Stein, R. Shakarchi, Fourier analysis: An introduction, Princeton Univ. Press, 2003.
