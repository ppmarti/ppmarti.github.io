Basic Linear Algebra

This is a learning resource page for Linear Algebra, mainly for Part 2 undergraduates.
It is not specific to any module or programme. Check your syllabus.

Selected Lecture Notes and Other Reading

The following sets of notes may provide useful support for your studies.
  • Some handy notes on LINEAR ALGEBRA
  • One good way to learn more about linear algebra (or to learn about it if the formal treatment doesn't suit you!) is to think about its applications. There are lots of interesting problems in Economics, Engineering , Physics, Finance, Chemistry and so on that are well modelled by a system of simultaneous linear differential equations. These are quite easy to solve if we can do linear algebra! For examples see the textbooks mentioned below.
